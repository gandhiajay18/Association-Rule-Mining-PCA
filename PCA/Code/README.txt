Programming platform used: Python 3

Steps on how to run the code:
1. Copy the entire PCA folder into a suitable location from where you want to run it.
2. Put the input files in the "input" folder which already have "pca_a.txt", "pca_b.txt","pca_c.txt", "pca_demo.txt" by default.
3. Run PCA.py on a compiler which have the libraries: numpy, matplotlib, sklearn.
4. Check the saved plots in the output folder.
